stm32pills
==========

These are minimal STM32CubeMX project files for the Chinese boards commonly known as *Blue Pill* and *Black Pill* -- MCU STM32F103C8Tx.

These projects were set up using STM32Cube firmware version FW_F1 V1.7.0.

Project files:
- stm32blackpill.ioc
- stm32bluepill.ioc
